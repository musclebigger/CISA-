    1.1.1.ISACA信息系统审计和鉴证标准
    首先具有ISACA的标准具有强制性，标准分为三类：通用、执行和报告。
        。通用:适用于所有任务的执行，指导原则
        。执行：不同任务如何执行与定义
        。报告：报告方式
